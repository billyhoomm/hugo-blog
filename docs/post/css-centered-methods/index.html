CSS几种居中方法解释

1.水平居中

行级元素设置其父元素的text-align center，块级元素设置其本身的left 和 right margins为auto即可，下面主要介绍下容器内的垂直居中和完全居中

2.line-height

单行文本垂直居中

1
2
3

<div id="parent">
  <div id="child">Text here</div>
</div>
1
2
3
4

#child {
  line-height: 200px;
  text-align:center;
}

图片垂直居中：

1
2
3

<div id="parent">
  <img src="">
</div>
1
2
3
4
5
6

#parent {
  line-height: 200px;
}
#parent img {
  vertical-align: middle;
}

3.table & table-cell

通用方法（vertical-align只对行内元素或者行内快起效，table-cell默认为行内块）

1
2
3

<div id="parent">
  <div id="child">Content here</div>
</div>
1
2
3
4
5

#parent {display: table;}
#child {
  display: table-cell;
  vertical-align: middle;
}
